## Installation

i use:  
nodejs version: v16.17.0  
npm: 8.19.2

add data.json file in backend folder, you can use example file example.json

open terminal

```bash
  cd backend
  npm install
  node index.js
```

open new terminal again

```bash
  cd frontend
  npm install
  npm start

```
